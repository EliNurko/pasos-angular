Códigos de Git

Códigos para guardar un git

  • git init: Sirve para crear el repositorio
  • git add .: Sirve para añadir los documestos credados en el repositorio
  • git commit -m: Sirve para guardar los cambios con el título
  • Git push origin master: Sirve para mandar el commit al repositorio online

Código para recuperar commits

  • Git restore –-hard: Restaura el commit
