Pasos para crear un input y output

Crear componentes

  • Crear un modulo de "components" y "pages" con --flat
  • Crear un componente padre y uno hijo para enlazarlo

Archivos app.component

  • Importar el componentsModule y pages.Module
  • Crear una ruta para el components "padre"

Arcivo componentsModule

  • En la sección de "declarations" colocar "CommonModule"
  • En la sección de exports colocar el componente "hijo"

Arcivo Pagesmodule

  • Importar el "CommonModule"
  • Importar el "ComponentsModule"
  • Colocar en declarations el componente "padre"

Componente hijo

  • Se coloca una función
  • Con el @-input (sin guión) se coloca la función que se desea importar al componente padre
  • El @-output (sin guión) nos cambia el valor en el componente padre
